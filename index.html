Introduction

Welcome to my Final Year Project on an Object Measurement System. This project aims to develop a system that can accurately measure objects using computer vision and machine learning techniques.

Project Objective

The main objective of this project is to design and implement a system that can measure the dimensions of objects in real-time. This system can be applied in various fields such as manufacturing, logistics, and healthcare.

Tools and Components

This project utilizes the following tools and components:

  • Programming Languages: Python, JavaScript, Html
  • Software: Telegram bot
  • Hardware: Raspberry Pi 4, Ultrasonic sensor, Load cell 200kg, LCD1602A

Steps to Run

Follow these steps to run the object measurement system:

  • Connect power supply
  • Put an object on the load cell
  • When the LCD shows "Measuring..." it means the system has detected an object.
  • The measurement output will display on the LCD and send to the Telegram bot.
